Chapter 8. Promoting Your Social Application

Creating a good application is not enough to succeed in the social platform world. It is also important to promote our application using different channels.

We should define our application's goal by identifying the types of users we want to reach, and working out the promotional plan.

Getting to know your users

The Facebook Platform provides a data analysis tool to understand the type of users and how they use it.

In order to access the Insights tool, visit Facebook App Dashboard and select our social application.

Click on Edit Settings or Edit App as shown in the preceding screenshot.

Click on Insights on the left side bar of the page. The following screenshot shows the Overview page of the Insights tool:

The Overview page provides information about our active users, as seen in the next screenshot. Using the period filter and the Export button, we can export our data to Excel or Comma-separated Values (CSV) file formats.

It is really important to understand how users interact with our application. Based on Insights, we can understand which features are more popular among our user pool. This provides useful feedback, so we can focus on improving the less popular features. The following screenshot shows the Sharing chart that describes activities on Facebook:

The Insights tool provides information that will help you to understand the way your application is engaging users.

I strongly recommend going over all the features provided by Insights and taking advantage of them while your application is live.

Growing your audience

Users are the most important resources for our application. The more users we have, the better. However, it is also important to grow our audience as much as possible.

We explored some of the following techniques to grow our audience:

  • Posting status updates through the application
  • Inviting Facebook friends to use our application
  • Using Facebook App Center

There are other techniques we can use to make our application viral. We should consider all available promotional channels provided by the Facebook Platform.

  • Canvas Page: This is a really important window shop for our application. As the application goes up in the ranking, Canvas Page will be visited more. Therefore, we should be careful about all the details listed on the page.
  • Suggest to Friends: This is a straightforward way to share your application with your friends straight from the Facebook web page.
  • Open Graph: This is a protocol to produce engaging stories on the Facebook Platform that gives developers the ability to customize the look and the feel as well as the content to engage users.

Mobile ads

Mobile ads are a new feature provided by the Facebook Platform to developers and application administrators to promote their applications on the Facebook News Feed. Using the Ads Manager tool, we are able to easily create multiplatform ads.

Note

https://developers.facebook.com/docs/tutorials/mobile-app-ads/

The ability to produce ads that can reach the right users gives us a powerful tool for promoting our mobile social applications.

In order to use mobile ads, we need to set up our social application. Open the LBAppDelegate.m file within your social application project. Identify the applicationDIdBecomeActive method. This method is called every time the application starts or resumes. We need to update that method code in order to support mobile ads. The next snippet contains the code that will enable our social application to use mobile ads:

- (void)applicationDidBecomeActive:(UIApplication *)application
{
  // Restart any tasks that were paused (or not yet started) while the application was inactive. If the application was previously in the background, optionally refresh the user interface.
  [FBSession.activeSession handleDidBecomeActive];
  [FBAppEvents activateApp];
}

First, we call the FBsession.activeSession handleDidBecomeActive method to resolve the session state for the Facebook's login flow.

The FBAppEvents activateApp method queries the Graph API to determine whether the application has an Install attribution enabled and publishes the ID in order to keep track of successful installations.

After the task code is complete, we can move on to create the real mobile ad that will appear on users' News Feed. Open Facebook Dashboard App, and select our current application and Edit App. Be sure Sandbox Mode is disabled, and iPhone App Store ID is set correctly. We can find our App Store ID on the Apple App Store using the URL template, https://itunes.apple.com/us/app/my-great-app/APP_STORE_ID.

Customize the previous URL using your App Store information.

On the left side bar, select App Details.

In the App Details page, be sure all necessary information is provided. We already took care of populating this form in the previous chapter. Ensure that Display Name and Primary Icons 75x75 are set. I strongly suggest populating as much information as possible to have more information in our mobile ads.

Now, we can start using Facebook's Ads Create tool.

https://www.facebook.com/ads/create

The following screenshot shows the Ads Create tool's landing page:

Within this page, you can advertise anything that exists on the Facebook Platform. In this case, we want to promote our application. Select iLikeBook and start creating the mobile ad.

In the following screenshot What would you like to do?, let us create a new mobile ad using the application information provided in the App Details page:

In this section, we can choose the type of campaign we want to move forward with. We have three different options:

  • Get New Users
  • Increase App Engagement
  • See Advanced Options

For the scope of this book, we are going to create the Get New Users campaign. Select the Get New Users radio button.

We can now customize the look and feel of our ad through the Your Ad section, for example, change title, description, and icon. On the right side of the section, we can have a preview of the ad. These types of ads will appear on the right side bar of Facebook Desktop App.

The Sponsored Stories section provides the ability to developers to create News Feed ads. The News Feed campaign is one of the most efficient because people are able to see which application their friends are using.

The next step of our campaign is to choose our audience. As best practice, I recommend spreading your audience as much as possible as test cases, slowly narrowing down to determine your audience target. The following screenshot shows the Audience section.

We can customize our campaign based on location, age, and interests of our audience.

Using the default option provided by the Platform, we have an audience of almost 160 million people. As we add more precise interests, the audience will decrease. The following screenshot shows the new audience number after defining the precise interest entry:

The new audience is about 14 million people. Once again, for testing purposes, I will strongly recommend you use a broader audience. If you are only trying to increase the number of users for your application with no audience target, just leave the default settings.

We now need to define the campaign information as follows:

  • Account information
  • Campaign name, schedule, and budget

As our next step, we need to review our ad and approve it. Once the mobile ad is approved, we will see our new ad showing up in News Feed and Facebook's right bar.

Mobile ads give developers the ability to promote application across the entire Facebook platform, from desktop computer to mobile devices using different ads layouts accordingly.

Summary

In this chapter, we learned how to promote our social application. First, we have to consider our target audience using the Insights tool provided by the Facebook Platform. We explored a few techniques in the previous chapter.

Mobile ads are a good tool to promote our application. We can set ads to target a specific set of people. For good practice, if you are going to set a mobile ad campaign, be sure you start with a large target. As you get to know your audience, slowly narrow down the target.

The mobile ads program offers different prices and options for our campaigns. Ultimately, with the largest target size, we can target the entire Facebook community.
