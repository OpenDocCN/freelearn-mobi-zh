So, what is BlueStacks?

BlueStacks is a suite of tools designed to allow you to run Android apps easily on a Windows or Mac computer. The following screenshot shows how it looks:

At the time of writing, there are two elements to the BlueStacks suite, which are listed as follows:

  • App Player: This is the engine, which runs the Android apps
  • Cloud Connect: This is a synchronization tool

As the BlueStacks tools can be freely downloaded, anyone with a PC running on Windows or Mac can download them and start experimenting with their capabilities. This book will walk you through the process of running BlueStacks on a computer and show you some of the ways in which you can make the most out of this emerging technology. There are other ways by which you can run an emulator of Android on your computer. You can, for instance, run a virtual machine or install the Android Software Development Kit (SDK). These assume a degree of technical understanding that isn't necessarily required with BlueStacks, making BlueStacks the quickest and easiest way of running apps on your computer.

BlueStacks is particularly interesting for users of Windows 8 tablets, as it opens up a whole library of mature software designed for a touch interface. This is particularly useful for those wanting to use many, free, or cheap Android apps on their laptop or tablet.

Note

It is worth noting that, at the time of writing this book, these tools are beta releases, so it is important that you take time to report the bugs that you may find to the developers through their website. The ongoing development and success of the software depends upon this feedback and results in a better product. If you become reliant on a particular feature, it is a good idea to highlight your love to the developers too. This can help influence which features are to be kept and improved upon as the product matures.

App Player

BlueStacks App Player allows a Windows or Mac user to run Android apps on his or her desktop or laptop. It does this by running an emulated version of Android within a window that you can interact with using your keyboard and mouse. The App Player can be downloaded and installed for free from the BlueStacks website, http://www.bluestacks.com. Currently, there are two main versions available for different operating systems:

  • Mac OS X
  • Windows XP, Vista, 7, and 8

Once you have installed the software, an Android emulator runs on your machine. This is a light version of Android that can access app stores so that you can download and run free and paid apps and content. Most apps are compatible with App Player; however, there are some that are not (for technical reasons) and some that have been prevented by the App developers from running.

Note

If you are running any another operating system on your computer, the more computing power you can make available to the App Player the better. Otherwise, you might experience slow loading apps or worse still ones that do not function properly.

To increase your chances of success, first try running App Player without running any other applications (for example, Word).

Cloud Connect

Cloud Connect provides a means to synchronize the apps running on an existing phone or tablet with the App Player. This means that you do not have to manually install lots of apps. Instead, you install an app on your device and sign up so that your App Player has exactly the same apps as your device.

Installation

You can install BlueStacks App Player and set it up on your system in three easy steps. The third step has been split up for Windows 7, 8, and Mac OS X so that you can follow the specific information for your operating system.

Step 1 – what do I need?

Before you install the BlueStacks App Player, you will need to check that you have all the required elements as listed here:

  • Disk space: 2 GB free (min).
  • Memory: 2 GB (min).
  • A Dual-core CPU (recommended).
  • BlueStacks App Player requires an OpenGL ES compatible graphics card and drivers (a utility such as DoctorGL can help identify your graphics card capability and check if driver upgrades may be required).
  • To install BlueStacks App Player on Windows, you need to be logged in with full admin rights. When installing on Mac OS X, you need to provide credentials, which have the rights to change the system.
  • Mac OS X installations require Snow Leopard or greater.

Top 5 features you need to know about

As you get more used to using BlueStacks, you'll start to appreciate the power of running Android apps on your computer that BlueStacks offers you. With that in mind, here are the top five features that will help you progress from being a novice so that you can make the most of the software.

The BlueStacks environment

Once you've got going with BlueStacks App Player, you will no doubt want to get under the hood and make some changes. The settings icon at the bottom right corner, as shown in the following screenshot, takes you to the page where changes to the environment are made:

When you are in the settings screen, as shown in the following screenshot, there are a couple of options you may wish to tweak that are described as follows:

  • Change Language: This option enables users who do not natively speak English to choose an option that is compatible with their mother tongues.
  • Manage Accounts: This option allows you to update your Google account for accessing the App Store. You can add more than one account if you have apps purchased in different accounts that you do not wish to purchase again.
  • Manage Applications: This option is where you can uninstall apps that you are no longer using and set whether they are the tablet or default versions of the app. The majority of the time the default option is fine, but you may need to select the tablet option if you are find that an app does not run automatically in the fullscreen mode.

1-Click Sync

This short guide will run you through the process of sending apps to your phone from BlueStacks App Player using 1-Click Sync, as listed in the following steps:

  1. Before starting, you will need to make sure that you have the same Google account setup in your Settings section as that on the device you wish to send an app to.
  2. From within App Player, click on the 1-Click Sync setup.
  3. Select the device you wish to sync the app to.
  4. Click on Done.
  5. Now click on the 1-Click Sync icon at the bottom right corner, as shown in the following screenshot, of the app you wish to send to your Android device:
  6. Now watch as the app starts automatically downloading.

Cloud Connect allows you to synchronize the apps that you have installed on your phone. The following steps help you synchronize your apps using Cloud Connect:

Apps from outside the app store

To install apps that do not appear in the app store, you will first need to get the APK file. This is the compiled Java application written specifically for Android. The processes are different for Mac and PC, so you'll need to follow the guide that relates to the type of computer you have.

Mac

Perform the following steps to install an app on your Mac that does not appear in the app store:

  1. Download an APK file from the internet, or if you are a developer, prepare one in your IDE. You can download one from http://www.freewarelovers.com/android/download/temp/1325762461_Zirco_Browser_0.4.3.apk.
  2. Once the file has downloaded, you need to copy it into this directory: Applications\BlueStacks.app\Contents\Runtime. If you are using Finder, you will need to right-click on the BlueStacks app and then click on Show Contents of Package.
  3. Open the terminal.
  4. Type the following commands:
      cd .. [Return]
  cd .. [Return]
  cd applications [Return]
  cd bluestacks.app [Return]
  cd contents [Return]
  cd runtime [Return]
  ./uHD-Adb install Zirco_Browser_0.4.3.apk [Return]

    When the process completes, you should receive a success message.

  5. This app will now be installed in your My Apps section of BlueStacks App Player.

PC

Perform the following steps to install an app on your PC that does not appear in the app store:

  1. Download an APK file from the internet, or if you are a developer, prepare one in your IDE. You can download one from http://www.freewarelovers.com/android/download/temp/1325762461_Zirco_Browser_0.4.3.apk.
  2. Right-click on the file and open with BlueStacks APK Handler.
  3. This app will now be installed in your My Apps section of the BlueStacks App Player

Tips for troubleshooting

If you find that you are running into trouble, you might want to consider the following tips for troubleshooting:

  • Your first troubleshooting tactic is to uninstall and reinstall BlueStacks. If it is beta software, you may find that the installation isn't successful the first time.
  • If you are experiencing a slow response from BlueStacks, you should try restarting your computer and make sure that very few apps are running in the background, as the App Player can be quite demanding on system resources.
  • If you are a Windows user, the next step is to install the latest drivers for your graphics card. If you find that the app fails to load, it is worth looking at the support forum to find out whether your graphics card has been used successfully by others.
  • If you are using Windows 8 Pro with a touch screen device and are not able to run App Player, try downloading and running the following registry patch: http://cdn.bluestacks.com/downloads/tools/fix-win8-pro-compat-rel.reg.

    You should take care before installing registry keys, as it poses a risk to your system. If in doubt, do not proceed, or if you wish to learn more, look at the following article from Microsoft: http://support.microsoft.com/kb/310516.
